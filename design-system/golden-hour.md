# Golden Hour — Design Palette

> Cálido, dorado, luminoso. Transmite confianza, creatividad y calidez humana.

## Colores base

| Nombre | Hex | Uso principal |
|--------|-----|---------------|
| Cream | `#FBF6F0` | Background principal |
| Ochre Gold | `#C8943C` | Secondary, badges |
| Soft Rose | `#EBBCB0` | Accent, detalles suaves |
| Burnt Sienna | `#C05835` | Primary, CTAs |
| Dark Cocoa | `#2C1810` | Texto principal |

## Roles semánticos (CSS Variables)

| Variable | Valor | Función |
|----------|-------|---------|
| `--color-bg` | `#FBF6F0` | Fondo general |
| `--color-surface` | `#FFFFFF` | Cards, nav |
| `--color-surface-2` | `#F0E8DC` | Hover, inputs |
| `--color-primary` | `#C05835` | Botones primarios, links |
| `--color-secondary` | `#C8943C` | Botones secondary |
| `--color-accent` | `#EBBCB0` | Detalles suaves, accent |
| `--color-text` | `#2C1810` | Texto principal |
| `--color-text-muted` | `#8B6B52` | Labels, captions |
| `--color-border` | `#E0D5C8` | Bordes, divisores |

## Graduaciones (100–1000)

Cada color expandido en 10 pasos (ver `index.html`).

## Colores semánticos (Status Badges)

| Estado | Color | Uso |
|--------|-------|-----|
| Error | `#DD4444` | Errores |
| Warning | `#D4930E` | Advertencias |
| Success | `#2E9B5F` | Éxito |
| Info | `#5B7EC2` | Información |

## Notas de uso

- Theme: **Light mode cálido**
- El Burnt Sienna como primary es audaz sin ser agresivo
- Contraste texto/fondo: alto (Dark Cocoa sobre Cream)
- Ideal para portfolios que buscan transmitir cercanía y profesionalismo
