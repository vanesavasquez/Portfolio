# Botanical Calm — Design Palette

> Natural, minimalista, orgánico. Evoca serenidad y conexión con lo esencial.

## Colores base

| Nombre | Hex | Uso principal |
|--------|-----|---------------|
| Linen | `#F5F0E8` | Background principal |
| Sand | `#C8B99A` | Secondary, badges |
| Sage Green | `#5C7050` | Primary, CTAs |
| Charcoal | `#3C3C3C` | Accent, elementos fuertes |
| Cream White | `#FDFCF8` | Superficies elevadas |

## Roles semánticos (CSS Variables)

| Variable | Valor | Función |
|----------|-------|---------|
| `--color-bg` | `#F5F0E8` | Fondo general |
| `--color-surface` | `#FDFCF8` | Cards, nav |
| `--color-surface-2` | `#EBE5D8` | Hover, inputs |
| `--color-primary` | `#5C7050` | Botones primarios, links |
| `--color-secondary` | `#C8B99A` | Botones secondary |
| `--color-accent` | `#3C3C3C` | Botón accent, detalles |
| `--color-text` | `#2C2C2C` | Texto principal |
| `--color-text-muted` | `#6B6B5F` | Labels, captions |
| `--color-border` | `#D8D0C2` | Bordes, divisores |

## Graduaciones (100–1000)

Cada color expandido en 10 pasos (ver `index.html`).

## Colores semánticos (Status Badges)

| Estado | Color | Uso |
|--------|-------|-----|
| Error | `#DD4444` | Errores |
| Warning | `#D4930E` | Advertencias |
| Success | `#2E9B5F` | Éxito |
| Info | `#5B7EC2` | Información |

## Notas de uso

- Theme: **Light mode natural**
- El Sage Green como primary aporta calma y sofisticación
- La paleta tiene baja saturación → diseño sereno y minimalista
- Ideal para portfolios que priorizan contenido sobre decoración
