# Terra Nova — Design Palette

> Terroso, sofisticado, audaz. Combina fuerza y elegancia orgánica.

## Colores base

| Nombre | Hex | Uso principal |
|--------|-----|---------------|
| Dark Olive | `#3A3B2F` | Accent, elementos profundos |
| Dark Gray | `#333333` | Texto principal |
| Dusty Rose | `#B58C91` | Secondary, badges |
| Warm Cream | `#F5F5E0` | Background principal |
| Burnt Orange | `#A84F00` | Primary, CTAs |

## Roles semánticos (CSS Variables)

| Variable | Valor | Función |
|----------|-------|---------|
| `--color-bg` | `#F5F5E0` | Fondo general |
| `--color-surface` | `#FCFCF0` | Cards, nav |
| `--color-surface-2` | `#E8E8CC` | Hover, inputs |
| `--color-primary` | `#A84F00` | Botones primarios, links |
| `--color-secondary` | `#B58C91` | Botones secondary |
| `--color-accent` | `#3A3B2F` | Botón accent, detalles |
| `--color-text` | `#333333` | Texto principal |
| `--color-text-muted` | `#5A5A50` | Labels, captions |
| `--color-border` | `#D0CCB0` | Bordes, divisores |

## Graduaciones (100–1000)

Cada color expandido en 10 pasos (ver `index.html`).

## Colores semánticos (Status Badges)

| Estado | Color | Uso |
|--------|-------|-----|
| Error | `#DD4444` | Errores |
| Warning | `#D4930E` | Advertencias |
| Success | `#2E9B5F` | Éxito |
| Info | `#5B7EC2` | Información |

## Notas de uso

- Theme: **Light mode terroso**
- El Burnt Orange como primary es audaz y con personalidad
- Dark Olive + Dusty Rose crean un contraste sofisticado
- La paleta tiene fuerte carácter → ideal para portfolios que buscan diferenciarse
